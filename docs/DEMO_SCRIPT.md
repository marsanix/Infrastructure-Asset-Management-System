[Demo Script — IAMS Sidang]

1. Buka https://localhost → Landing page
   - 3 scroll: Hero, Features, FAQ, CTA
   - Klik EN/ID → bahasa berubah realtime
   - Klik icon matahari/bulan → dark/light smooth transition

2. Klik "Mulai Sekarang" → Login
   - Klik "Administrator" di quick login
   - Masuk ke Dashboard

3. Dashboard
   - 4 KPI cards (Total Aset, Incident, Problem, Jaringan)
   - Distribusi aset per kategori
   - Status breakdown

4. Assets (sidebar kiri)
   - Tabel data 8 aset
   - Filter status/kategori
   - Klik "Tambah Aset" → isi form → Simpan
   - Serial Number wajib & unique (demo error jika duplikat)
   - Edit aset → ubah status → Perbarui
   - Hapus aset → confirmation dialog

5. Incidents
   - Tabel insiden + severity badge
   - Create new incident

6. Requests
   - Tabel request + status filter
   - Create request → pilih tipe request

7. Changes
   - Tabel change + risk/impact badge
   - Klik Approve → status berubah ke Approved
   - Klik Reject → status berubah ke Rejected

8. Master Data
   - 5 tab: Departments, Locations, Categories, Brands, Models
   - Create data → Edit → Delete
   - Demo delete safety: coba hapus data yang dipakai → "tidak bisa dihapus"

9. Users / Roles
   - Daftar user + role
   - Hapus user → hilang dari daftar

10. Reports
    - Warranty Watch
    - Export CSV

11. Audit Logs
    - Semua aksi terekam
    - Filter action/status

12. Logout → Login sebagai Operator
    - Sidebar hanya 6 menu (tanpa Master Data, Users, Reports, Audit)
    - Tidak bisa delete asset/request/change
    - Akses /master-data via URL → redirect ke dashboard

=== TIMING: 5-7 menit ===
