# Aplikasi Infrastructure Asset Management System (IAMS)

Teknologi:
- Aplikasi berbasis Web (Responsive)
- Microservices (Front-end: Vue 3 + Vite + shadcn-vue, Back-end: Python Flask, Postgresql, Docker, Docker-compose)
- Template front-end menggunakan shadcn ui vue (Vue 3 + Vite + shadcn-vue)
- Implementasi keamanan: OWASP Top 10:2025 dan MITRE ATT&CK framework.
- Multiple languages Support (English, Indonesian)
- Theme: Light, Dark, System
- Responsive Design (Mobile, Tablet, Desktop)

Fitur Aplikasi:
- Dashboard
- Manajemen Aset
- Manajemen Pengguna
- Manajemen Departemen
- Manajemen Lokasi
- Manajemen Kategori
- Manajemen Merek
- Manajemen Model
- Manajemen Jaringan    
- User Login / Logout (Authentication & Authorization)
- Role Management (Administrator, User)
- Role Based Access Control (RBAC)
- Change Management
- Incident Management
- Problem Management
- Request Management
- Report Management
- Audit Log

Fitur Tambahan:
- Secure authentication notes for assets, router, switch, firewall, AP, printers with encryption (bcrypt).

Struktur Database: (terlampir di file database.sql)
